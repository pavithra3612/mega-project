# ENG220 Combined Project Dashboard

This repository contains a **multi-team interactive Streamlit dashboard** developed for the ENG220 course. Each team contributed a unique data analysis and visualization module, all integrated into a single unified application.

---

## 🚀 Overview

The dashboard allows users to explore multiple datasets and projects through an easy-to-use sidebar navigation system. Each team focuses on different real-world topics such as:

- Environmental Justice Index (EJI)
- Air Quality (AQI)
- Gun Violence Analysis
- Climate Change & Natural Disasters
- Emissions & Walkability
- Financial Indicators
- Infrastructure & Public Data

---

## 🧭 How to Use

1. Launch the app.
2. Use the **sidebar navigation** to select a team project.
3. Each team page includes:
   - Data visualizations 📊
   - Interactive filters 🎛️
   - Insights and summaries 📄
